# Hangman Game

Terminal hangman game in C.

## Purpose and target
- Purpose: Implement a classic word-guessing game using strings and file I/O.
- Target: Learning project for game loops and input handling.

## What's inside
- hangman.c: game loop and UI
- words.c: word list handling and helpers
- hangman.exe: prebuilt Windows binary

## Build
```bash
gcc hangman.c words.c -o hangman
```

## Run
```bash
./hangman
```
